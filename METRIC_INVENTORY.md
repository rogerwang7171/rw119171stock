# 黄金坑策略 v4.0 统计标准清单（首轮梳理）

基于你提供的策略代码，当前用于筛选与排序的“统计标准/量化指标”如下。

## 1) 强度类（相对排名）

1. **RPS(120日)**
   - 定义：先计算每只股票120日涨幅，再在股票池内按涨幅排序，映射为百分位（0-100）。
   - 公式：`RPS = (rank + 1) / total * 100`
   - 用途：
     - 过滤：`RPS > 70`
     - 排序：候选结果按RPS降序输出。

## 2) 价格行为/趋势反转类

1. **20日回撤（Drawdown）**
   - 定义：当前收盘价相对近20日最高价的回撤。
   - 公式：`drawdown = current_close / high_20d - 1`
   - 阈值：`-15% ~ -8%`。

2. **MA5（5日均线）站上信号**
   - 定义：当前收盘价高于5日均线。
   - 公式：`ma5 = mean(close[-5:])`
   - 条件：`current_close > ma5`。

3. **10日低点稳定性（2天不破低）**
   - 定义：以10日最低价为参考，最近2天不继续创新低（含微小误差容忍）。
   - 关键参数：误差容忍`0.1%`（`*1.001`）。

## 3) 量能结构类

1. **前3日缩量比例（vol_shrink_ratio）**
   - 定义：前3日平均成交量与20日平均成交量之比。
   - 公式：`vol_shrink_ratio = avg_vol_3d_before / avg_vol_20`
   - 阈值：`< 0.6`。

2. **当日放量比例（vol_expand_ratio）**
   - 定义：当日成交量相对前3日平均成交量。
   - 公式：`vol_expand_ratio = today_vol / avg_vol_3d_before`
   - 阈值：`> 1.2`。

## 4) 基本面类

1. **净利润增速（profit_yoy / du_profit_rate）**
   - 定义：财务字段 `PERSHAREINDEX.du_profit_rate`。
   - 阈值：`> 10%`。
   - 处理：缺失数据时“放行并标记”。

## 5) 市场环境类（按股票所属指数）

1. **指数相对MA20位置（M1）**
   - 定义：指数当前值需高于其20日均线的一定比例。
   - 条件：`index_close > index_ma20 * 0.98`。

2. **指数5日涨跌幅（M2）**
   - 定义：指数近5日涨跌幅。
   - 公式：`change_5d = closes[-1] / closes[-5] - 1`
   - 阈值：`> -3%`。

## 6) 非“统计打分”但影响样本空间的规则

1. **股票池范围**：沪深300 + 中证500 + 中证1000 + 科创板。
2. **样本排除**：ST / *ST。
3. **输出建议**：固定建议金额10万/只，按股价折算为100股整数倍。

---

如果下一步你要改策略，我建议先明确：

1. 是要调整**阈值**（如 RPS>70 改为 >75）还是调整**指标定义**（如 RPS 120日改 60日+120日综合）。
2. 是否把“缺失基本面放行”改为“剔除”或“降权”。
3. 是否增加新统计标准（例如波动率、换手率分位、行业内RPS等）。
